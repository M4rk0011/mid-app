What is Data Binding?

Data binding is a powerful feature in Angular that allows communication between your TypeScript code (component logic) and your HTML template (user interface). It automatically synchronizes your data and keeps your UI in sync with the underlying data model.

Types of Data Binding

String Interpolation

{{ "{{ expression }}" }}

One-way data binding from component to template. Uses double curly braces to display dynamic data.

  • Display component properties
  • Evaluate expressions
  • Call component methods
  • Access object properties
View Examples

Property Binding

[property]="expression"

One-way data binding from component to DOM properties. Uses square brackets to bind to element properties.

  • Bind to DOM properties
  • Set element attributes
  • Control element states
  • Dynamic styling
View Examples

Key Differences

Feature String Interpolation Property Binding
Syntax {{ "{{ expression }}" }} [property]="expression"
Output Text content only DOM properties
Use Case Displaying dynamic text Setting element properties
